# Life Calendar

A simple, aesthetic, and customizable wallpaper generator to track your goals.

## Features

*   **Minimalist Aesthetic:** A clean and modern design that is easy on the eyes.
*   **Customizable:** Easily change the goals and title by editing the `goals.json` file.
*   **Vercel Integration:** Deploy your own wallpaper generator with a single click.

## How to Use

1.  **Edit `goals.json`:** Open the `goals.json` file and edit the `title` and `goals` to your liking.
2.  **Generate Wallpaper:** Run the `api/generate.py` script to generate a new wallpaper.
3.  **Set as Wallpaper:** Use the `client/update_wallpaper.py` script to set the generated wallpaper as your desktop background.

## Aesthetic

The wallpaper is generated with a minimalist aesthetic, using a clean and modern design. The color palette is simple and easy on the eyes, and the fonts are clean and readable. The goal is to provide a beautiful and functional wallpaper that helps you stay on track with your goals.
